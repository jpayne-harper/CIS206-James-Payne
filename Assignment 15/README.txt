Family Tree GUI Application (Python 3.7.3)

Instructions:
-------------
1. Unzip the folder and make sure both 'family_tree_gui.py' and 'family_tree.json' are in the same directory.
2. Run the program using Python 3.7.3:
   > python family_tree_gui.py
3. A window will appear where you can enter information for a family member.
4. Fields:
   - First Name (required)
   - Last Name (required)
   - Suffix (e.g., Jr., III, etc.)
   - Alive (Yes/No)
   - Father's Name
   - Mother's Name
   - Year of Birth (must be numeric)
5. Click "Add Member" to save the entry to the JSON file.
6. A confirmation popup will appear once the entry is saved successfully.
