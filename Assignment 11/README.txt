Northwind Database Interaction - Assignment 11
-------------------------------------------------
This ZIP includes two Python 3.7.3-compatible scripts for interacting with the Northwind SQLite database.

1. read_northwind.py
   - Lists all tables in the database
   - Prompts user to select one
   - Displays all records in the selected table
   - Formats results with row numbers and headers

2. modify_northwind.py
   - Lists tables and allows table selection
   - Lets user Insert, Update, or Delete records
   - Validates input, updates the database

Usage:
------
1. Make sure your Northwind SQLite file is saved (e.g., `Northwinds.db`)
2. Run either script from the terminal or IDE:
    > python read_northwind.py
    > python modify_northwind.py
3. When prompted, enter the full path to the database (e.g., `/Users/you/Desktop/Northwinds.db`)
